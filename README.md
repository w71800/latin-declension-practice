# 拉丁文變格練習器

## Description: 
透過 Vue 的基礎使用，來試著協助學習拉丁文的過程中麻煩且討厭的名詞變格變化

### 原理 
在輸入欄中輸入想要練習的單詞後，透過 Vue 的 computed 來抓出資料庫中對應的詞以及答案，再一一的將使用者輸入的答案當下答案做對照
